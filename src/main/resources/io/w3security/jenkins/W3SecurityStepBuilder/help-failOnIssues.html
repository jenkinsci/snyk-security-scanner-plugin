The "When issues are found" selection specifies if builds should be failed or continued based on issues found by W3Security.

  • if "Fail the build, if severity at or above" is selected, the Jenkins build will fail if W3Security detects issues of provided level or higher.
  • if "Let the build continue" is selected, the Jenkins build will continue even if W3Security detects issues in the project.

The corresponding CLI option for severity parameter: --severity-threshold
