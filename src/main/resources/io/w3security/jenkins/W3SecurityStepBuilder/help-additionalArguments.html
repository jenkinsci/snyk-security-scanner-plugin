Additional runtime arguments that will be used to invoke the W3Security CLI. See the W3Security CLI help page for more details.

Use the standalone double-dash -- to pass arguments to the build tool invoked by the W3Security CLI. For example:

  • -- -Pprofile -Dkey=value for Maven projects.
  • -- --configuration runtime for Gradle projects.
  • -- -Dkey=value for SBT projects.
